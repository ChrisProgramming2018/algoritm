Product Sum

Takes a "special" array and returns its product sum.

Def: special array
is  a non empty array that contains either integers or other "special arrays"

Inside the array the integers are added and multiplied by there level of depth

sample Input [5,2,[7,-1], 3, [6,[-13,8],4]]
sample Output 12 (5+2+2*(7-1) + 3 + 2 * (6 + 3 * (-13 + 8)
